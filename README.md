# Controle de Estoque em Python

Projeto simples desenvolvido em Python para simular o controle de estoque de um centro de distribuição.

# Funcionalidades
- Cadastro de produto
- Validação de categoria
- Verificação de estoque mínimo por categoria
- Aviso automático para solicitação de compra

# Categorias e Estoque Mínimo
- Alimentos: 50 unidades
- Bebidas: 75 unidades
- Limpeza: 30 unidades
